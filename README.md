# News AI - Fake News Detection System

## Project Overview

This is an AI-powered web application that classifies news as real or fake using an external AI API (Grok/OpenAI).

The system is built using Spring Boot and secured with JWT authentication. It provides REST APIs for news analysis, user authentication, and email notifications.

---

## Tech Stack

**Backend:**
- Spring Boot
- Spring Security (JWT)
- REST APIs

**AI Integration:**
- OpenAI / Grok API

**Other:**
- Spring Mail!
- Maven

---

## Features

- Analyze news content using AI
- Classify news as Real or Fake
- Secure login/signup using JWT
- Email notification support
- RESTful architecture
